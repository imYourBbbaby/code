爬虫资料：

1.安装scrapy
  pip install scrapy
2.创建爬虫项目
  scrapy startproject spider_project

  cd spider_project
  scrapy genderspider spider_name xxx.com

3.运行爬虫
 scrapy crawl spider_name

4.运行demo[scrapy_project]
  python run.py
